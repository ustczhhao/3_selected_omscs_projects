# Job Offer Comparison App

This repository contains a mobile application project for comparing job offers across compensation, benefits, and location-adjusted factors. The project was developed as a team-based software engineering project and is presented here to showcase application design, implementation, and end-to-end development workflow.

## Project Overview

Choosing between job offers often involves more than comparing salary alone. Differences in cost of living, bonuses, retirement contributions, and other benefits can make side-by-side comparison difficult. This project addresses that problem by providing an Android application that helps users evaluate and compare current and prospective job offers in a structured way.

The application supports entry and management of a current job, entry of multiple job offers, configuration of comparison weights, and direct comparison between selected jobs.

## Key Highlights

- Developed an Android app for structured comparison of job offers

## Core Functionality

The application supports:

- Entering and updating a current job
- Entering and storing multiple job offers
- Adjusting comparison settings and weight preferences
- Comparing selected offers side by side
- Persisting application data between runs
